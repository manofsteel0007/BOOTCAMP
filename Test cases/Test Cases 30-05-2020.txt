Beautiful game

Given an array A of size N. The elements of the array consists of positive integers. You have to find the largest element with minimum frequency.

Input Format: First line of input contains number of testcases T. For each testcase there will be two lines. First line contains N, next line contains N elements separated by spaces.

Output Format: For each testcase, print the largest element with minimum frequency.

User Task:
Your task is to write a function LargButMinFreq(A, n) which accepts array Aand n. Hence you have to return the largest element with minimum frequency. If there are more than one elements with same frequency, you will have to return the largest element.

Constraints:
1 <= T <= 100
1 <= N <= 105
1 <= A[i] <= 106

Example:
Input 1: 
1
5
2 2 5 50 1

Output 1:
50



Input 2: 

1

4

2 2 4 4

Output 2:

4

Explanation :
Testcase 1: All elements are having frequency 1 except 2. 50 is the maximum element with minimum frequency.

Testcase 2: Both 2 and 4 occur twice i.e., same frequency. Therefore 4 is returned since it is the largest inspite of equal frequencies.
